# Pi Gateway 后台管理模块 PRD

## 1. 背景

Pi Gateway 现在主要通过飞书里的 `/命令` 管理会话、模型、设置、工具、定时任务、群聊规则和技能。

随着命令变多，在飞书里输入命令不够方便。尤其电脑就在身边时，管理员更希望打开本机网页，通过按钮、下拉框、开关和输入框完成同样操作。

## 2. 产品目标

做一个轻量后台网页，作为飞书 `/命令` 的快捷操作入口。

核心目标：

- 少打字
- 不改变现有命令能力
- 私聊和群聊分开操作
- 重要功能都有明确页面
- 页面保持简单，不做复杂后台系统

## 3. 非目标

第一版不做：

- 不做总览大盘
- 不做复杂数据统计
- 不做用户管理系统
- 不做定时任务新建表单
- 不做聊天记录管理
- 不做飞书通讯录全量管理
- 不做公网后台能力
- 不做统一命令页
- 不做临时目标 ID 输入
- 不做工具预设管理

## 4. 使用对象

主要使用者是网关管理员。

典型场景：

- 在电脑旁快速切换模型
- 查看某个私聊或群聊的会话状态
- 停止当前正在执行的任务
- 查看某个群的定时任务并运行、停止或删除
- 查看或调整群聊触发规则
- 管理当前目标里的工具开关
- 调整当前目标的消息和语音相关设置
- 查看技能列表和技能统计

## 5. 信息架构

登录后的后台由两部分组成：

- 左侧：具体功能页面
- 顶部：当前执行目标

左侧功能页面：

- 会话
- 模型
- 设置
- 定时任务
- 群聊设置
- 工具
- 控制
- 技能

顶部不放产品标题栏，不放功能页签。

## 6. 设计原则

### 6.1 先选目标，再操作

顶部始终显示当前执行目标。

执行目标只支持从已知目标里选择：

- 私聊
- 群聊

选择目标后立即切换，不需要再点确认按钮。

选定目标后，页面里的所有操作都作用在这个目标上。

### 6.2 私聊和群聊必须隔离

同一个命令，在不同目标里结果可能不同。

例如：

- `/sessions`：不同私聊、不同群聊的会话列表不同
- `/cron list`：不同目标的定时任务不同
- `/group`：只对当前群聊有意义
- `/tools`：不同会话的工具状态可能不同

后台必须明确显示当前目标，避免误操作。

### 6.3 常规设置即时生效

常规设置修改后立即应用，不提供“保存设置”或“应用更改”按钮。

例如：

- 切换模型
- 开关模型路由
- 修改消息流式输出
- 修改语音识别开关
- 修改群聊触发规则
- 修改关键词
- 开关工具

只有明确的一次性动作保留按钮，例如执行命令、立即运行任务、停止任务、删除任务、清空关键词、加入或移出群白名单。

### 6.4 页面只放当前操作需要的内容

页面文案必须直接服务当前操作。

不展示：

- 产品介绍
- 原型说明
- 实现背景
- 研发备注
- 下一步建议
- 多余提示
- 重复导航

### 6.5 保留原始命令入口

每个功能页保留一个“原始命令”输入框，用来执行该页面对应的命令。

这样即使某个参数暂时没有专门控件，也可以直接执行原始命令。

示例：

```text
/model
/cron list
/group keywords show
/tools reset
```

## 7. 通用顶部目标栏

用途：选择当前要操作的私聊或群聊。

内容：

- 当前目标名称
- 当前目标类型
- 已知目标选择框
- 是否同步结果到飞书

交互：

- 选择框里只展示已知目标
- 选中目标后立即切换
- 不提供单独的“切换”按钮
- 不提供临时 ID 输入模式
- 当前目标为空时，不允许执行命令

目标来源：

- 历史私聊
- 历史群聊
- 管理员配置
- 群白名单
- 定时任务里出现过的目标

## 8. 页面范围

### 8.1 登录页

用途：防止本机后台被随便访问。

内容：

- 管理密码输入框
- 登录按钮

登录页只保留登录所需内容，不展示产品说明或命令示例。

第一版只需要简单密码即可。

### 8.2 会话页

用途：查看当前目标的会话。

功能：

- 查看当前状态
- 查看历史会话
- 查看已加载上下文

不做：

- 不做新建会话
- 不做重置会话
- 不做恢复历史会话
- 不接入任何会改变会话状态的操作

对应命令：

```text
/status
/sessions
/context
```

### 8.3 模型页

用途：查看和切换当前目标的模型设置。

功能：

- 查看当前模型
- 查看可用模型
- 选择 router / light / heavy 模型
- 开关模型路由

交互：

- 模型选择后立即应用
- 路由开关切换后立即应用
- 不提供保存按钮

对应命令：

```text
/model
/models
/model router <模型>
/model light <模型>
/model heavy <模型>
/route on
/route off
```

### 8.4 设置页

用途：管理当前目标的通用设置。

功能：

- 查看当前设置
- 开关流式输出
- 开关语音识别
- 开关表情回应
- 开关工具调用展示
- 查看或设置技能目录

交互：

- 开关切换后立即应用
- 技能目录修改后立即应用
- 不提供保存按钮

对应命令：

```text
/settings
/stream on
/stream off
/stt on
/stt off
/reaction on
/reaction off
/toolcalls on
/toolcalls off
/skill-folder
/skill-folder <路径>
```

### 8.5 定时任务页

用途：查看和控制当前目标的定时任务。

第一版只做：

- 查看任务列表
- 立即运行任务
- 停止正在运行的任务
- 删除任务

不做：

- 不做新建任务
- 不做复杂编辑表单

对应命令：

```text
/cron list
/cron run <jobId>
/cron stop <jobId>
/cron remove <jobId>
```

### 8.6 群聊设置页

用途：管理当前群聊的触发规则。

只在目标是群聊时显示。

功能：

- 查看群聊设置
- 设置群聊策略
- 设置触发方式
- 查看关键词
- 设置关键词
- 清空关键词
- 查看白名单
- 当前群加入或移出白名单

交互：

- 群聊策略修改后立即应用
- 触发方式修改后立即应用
- 关键词修改后立即应用
- 清空关键词保留独立按钮
- 加入或移出白名单保留独立按钮

对应命令：

```text
/group
/group policy disabled|allowlist|open
/group mode mention|all|keyword
/group keywords show
/group keywords set <关键词>
/group keywords clear
/group allowlist show
/group allowlist add here
/group allowlist remove here
```

### 8.7 工具页

用途：管理当前目标会话里的工具开关。

功能：

- 查看工具列表
- 启用工具
- 禁用工具

交互：

- 工具开关修改后立即应用
- 不做工具预设管理
- 不提供恢复默认专区

对应命令：

```text
/tools
/tools on <tool>
/tools off <tool>
```

补充：

如果需要执行 `/tools reset`，可以在工具页的原始命令输入框里执行，不做单独界面。

### 8.8 控制页

用途：控制当前目标正在执行的任务或运行状态。

功能：

- 停止当前任务
- 继续下一个排队任务
- 重启网关

对应命令：

```text
/stop
/next
/restart
```

### 8.9 技能页

用途：查看当前可用技能和技能统计。

功能：

- 查看技能列表
- 查看技能目录
- 查看技能使用统计

对应命令：

```text
/skills
/skillstat
```

## 9. 关键交互

### 9.1 执行命令

流程：

1. 用户选择执行目标
2. 用户进入具体功能页
3. 用户点击页面里的动作按钮，或在原始命令输入框里执行命令
4. 页面显示执行结果
5. 如用户打开“同步结果到飞书”，结果也发回对应私聊或群聊

设置类操作不走这个流程，修改后立即应用。

### 9.2 未选择目标

如果没有选择目标，不允许执行命令。

页面提示：

```text
请先选择私聊或群聊。
```

### 9.3 群聊命令限制

当目标不是群聊时，不显示群聊设置页。

如果用户通过原始命令执行 `/group`，提示：

```text
请先选择一个群聊。
```

### 9.4 目标不存在

如果目标不在已知列表里，第一版不支持在后台临时添加。

需要先让目标进入历史记录、管理员配置、群白名单或定时任务目标来源。

## 10. 第一版验收标准

第一版完成后，应满足：

- 可以登录后台
- 登录页只展示登录表单
- 左侧导航展示具体功能页
- 顶部目标栏可以选择已知私聊或群聊
- 选择目标后立即切换
- 不提供目标确认按钮
- 不提供临时目标 ID 输入
- 不提供统一命令页
- 会话、模型、设置、定时任务、群聊设置、工具、控制、技能页可用
- 会话页只提供查看能力
- 定时任务页支持查看、运行、停止、删除
- 不提供定时任务新建入口
- 不提供总览页
- 不提供工具预设管理
- 常规设置修改后立即生效，不需要保存按钮
- 每个功能页保留原始命令输入框
- 执行结果能在页面显示
- 可选同步结果到飞书
