# Overview

## What is Feishu Store App?

Feishu Store App is a software service created by Independent Software Vendors (ISVs) using Feishu Suite's open capabilities. It is accessed by businesses through the Feishu [App Directory](https://app.feishu.cn/).

Store Apps are developed by ISVs for general product applications, providing a one-to-many distribution channel through Feishu. Feishu's official review is required before launching a Store App in the app directory.

Enterprise administrators and users can easily discover and install applications from the Feishu App Directory. As an application developer, this directory can help you gain enterprise customers and revenue.

## Prerequisites

Before applying for store apps, it's important to understand the different [application types](https://open.feishu.cn/document/home/app-types-introduction/overview). Determine if store apps are suitable based on [the application store admission standards](https://open.feishu.cn/document/uMzNwEjLzcDMx4yM3ATM/uUzNwEjL1cDMx4SN3ATM).

If not satisfied, please first serve customers and validate the product througha custom app. Refer to the details of [custom apps development process](https://open.feishu.cn/document/home/introduction-to-custom-app-development/self-built-application-development-process).

The development and listing process for a store app is as follows.

1. **Settlement and verification**: Any organization wishing to develop a store app and list it in the [Feishu App Directory](https://app.feishu.cn/) must pass ISV verification. Refer to [ISV Entry](https://open.feishu.cn/document/uMzNwEjLzcDMx4yM3ATM/uUzNwEjL1cDMx4SN3ATM) for details.

2. **Develop an app**: Developers create a store app in the [Developer Console](https://open.feishu.cn/app) and complete frontend and backend development.

3. **Listing review**: The listing of Store apps requires strict review by the official Feishu operation. Refer to [Develop and List Store Apps](https://open.feishu.cn/document/uMzNwEjLzcDMx4yM3ATM/uYzNwEjL2cDMx4iN3ATM) and [ISV Daily Operational Management Specifications](https://open.feishu.cn/document/uAjLw4CM/uMzNwEjLzcDMx4yM3ATM/isv-management-specifications) for more information.

4. **Installation and use**: After the store app passes the listing review, any Feishu tenant can find and install the application in the [Feishu App Directory](https://app.feishu.cn/). After the tenant admin authorizes the application in the application market, this application will appear on the Feishu workplace of the enterprise members, and they can start using it.


![](https://sf3-cn.feishucdn.com/obj/open-platform-opendoc/81855762674e6fcd1b1ffaf8dc925a0c_Kq0TWXh2s1.png?height=678&lazyload=true&width=1640)

**Learn more**

- To install a store app, a tenant must go through the review and configuration process by the tenant admin in the [Feishu Admin Console](https://admin.feishu.cn/).
- The tenant admin is responsible for reviewing and approving application installation and usage requests from regular members.
- A store app can be installed in multiple tenants.
- Store apps need to hold the enterprise's [access token](https://open.feishu.cn/document/home/quickly-develop-three-party-approvals/creating-applications-and-requesting-permissions) , and then access the enterprise's data on Feishu with the authorization credentials.
